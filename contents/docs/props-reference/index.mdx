## Advanced Settings

Configure table behavior via `advancedSettings` prop:

| Setting                 | Type      | Description                         | Learn More                                   |
| ----------------------- | --------- | ----------------------------------- | -------------------------------------------- |
| `marginTop`             | `string`  | Top margin of table                 | [Advanced Settings](/docs/advanced-settings) |
| `noBorder`              | `boolean` | Remove table borders                | [Advanced Settings](/docs/advanced-settings) |
| `pinColumns`            | `boolean` | Pin columns during scroll           | [Advanced Settings](/docs/advanced-settings) |
| `alignCenterInLine`     | `boolean` | Vertically center content           | [Advanced Settings](/docs/advanced-settings) |
| `currentSortId`         | `string`  | Currently sorted column ID          | [Advanced Settings](/docs/advanced-settings) |
| `contrastRow`           | `boolean` | Alternate row background colors     | [Advanced Settings](/docs/advanced-settings) |
| `indexUseOriginalOrder` | `boolean` | Show original indices when filtered | [Advanced Settings](/docs/advanced-settings) |
| `enableRowDragDrop`     | `boolean` | Enable row reordering               | [Advanced Settings](/docs/advanced-settings) |
| `currentUsername`       | `string`  | Current user for comments           | [Advanced Settings](/docs/advanced-settings) |
| `showIndex`             | `boolean` | Display row numbers                 | [Advanced Settings](/docs/advanced-settings) |
| `selectNewOptionOnAdd`  | `boolean` | Auto-select new list options        | [Advanced Settings](/docs/advanced-settings) |
| `canAddListOptions`     | `boolean` | Allow adding options to lists       | [Advanced Settings](/docs/advanced-settings) |
| `showFooter`            | `boolean` | Display table footer                | [Advanced Settings](/docs/advanced-settings) |
| `rowIdProperty`         | `string`  | Unique row identifier property      | [Advanced Settings](/docs/advanced-settings) |
| `tableId`               | `string`  | Table instance identifier           | [Advanced Settings](/docs/advanced-settings) |
| `focusedRowIndex`       | `number`  | Row to highlight                    | [Advanced Settings](/docs/advanced-settings) |
| `showFlashOnCellUpdate` | `boolean` | Flash animation on cell update      | [Advanced Settings](/docs/advanced-settings) |
| `disabledRows`          | `object`  | Disable rows by property values     | [Advanced Settings](/docs/advanced-settings) |

**\* Required prop**
