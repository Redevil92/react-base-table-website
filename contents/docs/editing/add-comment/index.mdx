## Add a Comment

Enable users to leave notes on cells for team collaboration.

### Enable Comments

```tsx
<BaseTable
  headers={headers}
  items={items}
  comments={comments}
  advancedSettings={{
    currentUsername: "john.doe",
  }}
  onSaveComment={(comment, item) => {
    // Save comment to database
    saveComment(comment)
  }}
/>
```

### Comment Structure

```tsx
interface CommentData {
  id: string
  headerId: string // Column ID
  itemId: string | number // Row ID
  text: string // Comment content
  author: string // Who wrote it
  timestamp: Date // When it was written
}
```

### Example

```tsx
const [comments, setComments] = useState<CommentData[]>([])

<BaseTable
  headers={headers}
  items={items}
  comments={comments}
  advancedSettings={{
    currentUsername: currentUser.name,
  }}
  onSaveComment={async (comment, item) => {
    try {
      const savedComment = await saveCommentToDatabase(comment)
      setComments([...comments, savedComment])
      showSuccess("Comment saved")
    } catch (error) {
      showError("Failed to save comment")
    }
  }}
/>
```

### Manage Comments

Track and manage comments:

```tsx
const getCommentsForCell = (headerId: string, itemId: string | number) => {
  return comments.filter((c) => c.headerId === headerId && c.itemId === itemId)
}

const deleteComment = async (commentId: string) => {
  await deleteCommentFromDatabase(commentId)
  setComments(comments.filter((c) => c.id !== commentId))
}
```
