## Editing

Master all editing capabilities in react-base-table.

### Editing Workflow

1. **Define editable columns** with `editOptions`
2. **Choose editor type** - text, number, or select
3. **Handle changes** with callbacks
4. **Validate** input with conditional rules
5. **Persist** updates to database

### Key Concepts

| Concept           | Description                           |
| ----------------- | ------------------------------------- |
| **Cell Editor**   | UI component for editing a cell value |
| **Single Update** | Edit and save one cell                |
| **Bulk Update**   | Edit multiple cells at once           |
| **Context Menu**  | Right-click options for editing       |
