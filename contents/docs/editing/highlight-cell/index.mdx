## Highlight a Cell

Apply conditional styling to cells for visual organization.

### Basic Highlighting

```tsx
<BaseTable
  headers={headers}
  items={items}
  highlightCondition={[
    {
      propertyId: "status",
      value: "in progress",
      style: { backgroundColor: "#ffeb3b", color: "#000" },
      columnId: "status",
    },
  ]}
/>
```

### HighlightCondition Structure

```ts
import { type CSSProperties } from "react"

export default interface HighlightCondition {
  propertyId: string
  value: unknown
  style: CSSProperties
  columnId?: string // Optional, if provided, applies the style to the specific column only
}
```

### Highlight Callbacks

The table provides two callbacks for managing highlights:

```ts
onSetHighlightCondition?: (
  highlightCondition: HighlightCondition,
  item: TableItem
) => Promise<void> | void;

onRemoveHighlightCondition?: (
  highlightCondition: HighlightCondition,
  cssPropertyToRemove: keyof CSSProperties,
  item: TableItem
) => Promise<void> | void;
```

### Set Highlight

Allow users to highlight cells:

```tsx
<BaseTable
  headers={headers}
  items={items}
  highlightCondition={highlights}
  onSetHighlightCondition={(highlight, item) => {
    // Add highlight
    const newHighlights = [...highlights, highlight]
    setHighlights(newHighlights)
    saveHighlight(highlight)
  }}
  onRemoveHighlightCondition={(highlight, cssProperty, item) => {
    // Your logic here
  }}
/>
```

**Important:** To highlight only a single cell, ensure that the pair (`propertyId`, `value`) uniquely identifies that cell. A common pattern is to use `propertyId: "id"` with the item's unique id as the `value`. For example:

```tsx
{
  propertyId: "id",
  value: 123, // Unique item ID
  style: { backgroundColor: "#ffeb3b" },
  columnId: "status",
}
```
