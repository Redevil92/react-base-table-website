## Context Menu

The table provides built-in context menus that appear when users right-click on cells or headers. These menus offer quick access to common actions without cluttering the interface.

### Configuration

Enable or disable context menus through the `advancedSettings` prop:

```ts
advancedSettings={{
  contextMenu: {
    showOnHeader: boolean;  // Enable context menu on header cells
    showOnCell: boolean;    // Enable context menu on data cells
  }
}}
```

### Cell Context Menu

When right-clicking on a data cell, users can access cell-specific actions:

![Cell Context Menu](/images/context-menu/cell-context-menu.png)

Available actions:

- **Add a comment** — Leave a note on the cell for collaboration
- **Set color** — Apply a highlight color to the cell

### Header Context Menu

When right-clicking on a column header, users can access column-level operations:

![Header Context Menu](/images/context-menu/header-context-menu.png)

Available actions:

- **Sort** — Sort the table by this column
- **Choose columns** — Select which columns to display
- **Reset columns** — Restore columns to their default configuration
- **Chart range** — Create a chart from the selected data range
