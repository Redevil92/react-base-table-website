## Selection Context Menu

Right-click on selected rows or cells to perform bulk actions.

### Implementation

```tsx
const handleSelectionContextMenu = (
  e: React.MouseEvent,
  selectedIndices: number[]
) => {
  e.preventDefault()

  showContextMenu({
    x: e.clientX,
    y: e.clientY,
    items: [
      { label: "Copy", onClick: () => copySelection(selectedIndices) },
      { label: "Cut", onClick: () => cutSelection(selectedIndices) },
      { divider: true },
      {
        label: "Delete",
        onClick: () => deleteSelection(selectedIndices),
      },
      { divider: true },
      { label: "Export", onClick: () => exportSelection(selectedIndices) },
      {
        label: "Change status",
        submenu: [
          { label: "Active", onClick: () => updateSelection("Active") },
          { label: "Inactive", onClick: () => updateSelection("Inactive") },
        ],
      },
    ],
  })
}
```

### Available Actions

| Action    | Description                 |
| --------- | --------------------------- |
| Copy      | Copy all selected cells     |
| Cut       | Copy and mark for deletion  |
| Paste     | Paste to all selected cells |
| Delete    | Delete all selected rows    |
| Export    | Export selected rows as CSV |
| Bulk edit | Edit all at once            |
| Tag/Label | Add tag to selection        |

### Bulk Operations

```tsx
const handleBulkStatusChange = async (status: string, indices: number[]) => {
  const changes = indices.map((i) => ({
    itemUpdated: { ...items[i], status },
    originalIndex: i,
  }))

  try {
    await onBulkFieldChange(changes, "status")
    showSuccess(`Updated ${indices.length} items`)
  } catch (error) {
    showError("Failed to update items")
  }
}
```
