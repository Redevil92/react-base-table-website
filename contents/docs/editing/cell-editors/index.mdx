### Quick Reference

```tsx
// Text Editor
{
  editOptions: {
    editable: true,
    type: "string",
    defaultValue: "",
  },
}

// Number Editor
{
  editOptions: {
    editable: true,
    type: "number",
    defaultValue: 0,
  },
}

// Select Editor
{
  editOptions: {
    editable: true,
    type: "list",
    options: ["Option 1", "Option 2"],
    defaultValue: "Option 1",
  },
}
```

### Common Features

All editors support:

- Required field validation
- Conditional disabling
- Default values
- Custom styling
