## Selection

Enable users to select individual rows or multiple rows for batch operations.

### Basic Selection

Row selection allows users to:

- Select individual rows
- Select multiple rows
- Perform actions on selected rows

### Configuration

Selection features work with other table features like:

- Context menus for selected rows
- Bulk operations
- Highlighting

### Use Cases

- **Export** - Select and export specific rows
- **Bulk Actions** - Edit, delete, or update multiple rows
- **Organization** - Mark related items for grouped operations
