## Core Features Overview

react-base-table provides a rich set of features for building powerful data tables. Here's what you can do:

### Data Management

- **Column Definition** - Configure columns with types, validation, and custom rendering
- **Filtering** - Filter rows by column values
- **Sorting** - Sort by any column with custom sort logic
- **Selection** - Select single or multiple rows

### Editing

- **Text Fields** - Edit string values
- **Number Fields** - Edit numeric values with validation
- **Select Dropdowns** - Choose from predefined options
- **Single Cell Update** - Edit individual cells
- **Bulk Update** - Modify multiple cells at once
- **Copy and Paste** - Clipboard support for fast data entry

### Interaction

- **Drag & Drop** - Reorder rows by dragging
- **Context Menu** - Right-click menus for cells, selections, and headers
- **Highlighting** - Conditionally highlight cells
- **Comments** - Add notes to cells for collaboration

### Organization

- **Grouping** - Group rows by column values
- **Nested Rows** - Display hierarchical data

### Customization

- **Advanced Settings** - Fine-tune table behavior
- **Custom Rendering** - Override default cell and header rendering
- **Event Handlers** - Respond to user interactions
